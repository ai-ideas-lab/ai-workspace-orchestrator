# 2026-04-23 论文笔记

**论文标题：** An Attention-based Feature Extraction Method for Medical Image Classification

**核心方法：** 
提出MIAFEx方法，通过可学习的改进机制增强Transformer编码器中的分类token，提升医学图像特征提取质量。该方法能够有效处理医学图像数据的小样本量或高类内方差特性。

**主要结论：**
在多个复杂医学成像数据集上，MIAFEx在准确性和鲁棒性方面均优于传统特征提取器、CNN和ViT模型。特别在训练数据有限的场景下，该模型展现了更好的泛化能力，为医学图像分类提供了新的有效解决方案。
